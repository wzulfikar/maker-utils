# SaaS Maker Utils

A collection of utility functions for building SaaS applications in TypeScript/JavaScript.

## Installation

```bash
npm install saas-maker
```

Or with yarn:

```bash
yarn add saas-maker
```

Or with pnpm:

```bash
pnpm add saas-maker
```

## Core Utilities

SaaS Maker Utils provides a set of utility functions organized into the following categories:

- [Error Handling](/docs/error-handling) - Utilities for handling errors in your application
- [Validation](/docs/validation) - Functions to validate inputs and throw errors when validation fails
- [Logging](/docs/logging) - Logging utilities for consistent logging across your application
- [Result Type](/docs/result-type) - A result type for handling success and failure cases
